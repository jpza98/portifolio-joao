# Logos dos Clientes

Coloque aqui as imagens dos logos dos bancos e empresas.

## Arquivos necessários:

- `caixa.png` - Logo da Caixa Econômica Federal
- `bradesco.png` - Logo do Bradesco
- `santander.png` - Logo do Santander
- `itau.png` - Logo do Itaú
- `brb.png` - Logo do BRB
- `correios.png` - Logo dos Correios
- `cnh.png` - Logo da CNH

## Formato recomendado:
- PNG com fundo transparente
- Tamanho: aproximadamente 200x100px
- Formato: PNG ou SVG

Se as imagens não estiverem disponíveis, o site usará URLs externas como fallback.
